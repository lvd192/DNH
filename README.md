# Dự Án Tích Hợp Dữ Liệu & AI Chatbot - Dược Nam Hà

Thư mục này chứa mã nguồn và tài liệu kỹ thuật cho hệ thống Portal, Dashboard, Auto-Notification và AI Chatbot của Dược Nam Hà (DNH).

## Cấu Trúc Thư Mục Đề Xuất
* `docs/`: Tài liệu đặc tả yêu cầu, thiết kế database và API.
* `backend/`: Mã nguồn của API Middleware (Python/NodeJS) kết nối database trung gian.
* `frontend/`: Mã nguồn giao diện Webapp Portal & Dashboard (Next.js/React).
* `scripts/`: Các script ETL đồng bộ dữ liệu từ Bravo/DMS về database trung gian.
* `ai_agent/`: Mã nguồn tích hợp AI Chatbot (Text-to-SQL + Semantic Layer).
